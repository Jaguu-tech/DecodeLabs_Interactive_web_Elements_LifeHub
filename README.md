# LifeHub Pro - Student Productivity Dashboard
LifeHub Pro is an all-in-one productivity dashboard designed to help students manage their academics, finances, goals, travel planning, and daily activities from a single platform.
Built using HTML, CSS, and JavaScript, this project combines multiple productivity tools into one responsive and interactive web application.

## Featuresl

### Study Hub
* Pomodoro Timer
* Motivational Quote Generator
* Study Progress Tracker

### Student Dashboard
* Subject Manager
* Marks Calculator
* Percentage Calculator
* CGPA Calculator

### Daily Goals Tracker
* Add Goals
* Mark Goals as Completed
* Delete Goals
* Achievement Badges

### Notes Section
* Quick Notes
* Local Storage Support

### Travel Budget Planner
* Destination Selection
* Budget Calculation
* Expense Tracking
* Remaining Budget Display

### Mood Journal
* Daily Mood Tracking
* Journal Entries
* Local Storage Support

### Finance Tracker
* Income Management

### Water Intake Tracker

### Additional Features
